# Device Configurator Framework
## Reflections | Ben Smogolski


At the time of writing I have not received any peer review. This is unfortunate since I provided two other students honest feedback very quickly. However, I can comment on the instructor’s preliminary feedback:

Reframe slightly for an instructional lab

I understand the concern here and made sure that the exercise is performed within the familiar lab context(s), while keeping technical complexity to a minimum. While the lab has elements of a software design project (which is important for real world applications), it is intentionally limited and can be completed within a reasonable amount of time.

Tighten the scope

Scope creep is a concern with this lab, but I clearly indicated that certain modules were optional if the student faces time constraints. If a copy/paste approach is used, I have no concerns that the lab can be completed within 75 minutes. Only one menu function is required to demonstrate functionality. The two peer labs I reviewed were too short and probably too simple.

Clarify networking tasks

The student lab guide clearly indicates which functions and modules are implemented, while providing screenshots as reference outputs. This lab emphasises the importance of supporting frameworks to bring simple scripts to life. The networking code is simple by design and almost unimportant compared to the lab’s larger objective.

Define outputs and success criteria

A Success Indicators heading was added to the top of the student lab guide. Reference screenshots are included throughout the guide. The entire section “Use the Configurator Application” demonstrates successful outputs.

Overall Reflections

This capstone project was very challenging because there is a lot of accountability. The project is multidimensional, blending technical (Python, Git/DevOps, APIs) with nontechnical project management and writing. As a Git novice, using Git was a substantial hurdle, especially when switching context between the Linux CLI and the Github GUI. This is easily one of the most demanding academic projects I’ve ever undertaken.
